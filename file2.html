HTML and CSS Basics Summary

HTML Summary

HTML is used to define the page content not for styling the page, use CSS instead of that.

  • HTML is a Hyper Text Markup Language
  • HTML uses elements to describe contents
  • HTML typically have opening tag,content and then closing tag
  • you can also have void elements like images
  • you can also configure elements with attributes

Learn more about all HTML elememts on MDN HTML Reference

CSS Summary

CSS is used to style your page content

  • CSS is Cascading Style Sheet
  • Styles are assigned via property-value pairs
  • You can assign styles via the {style} attribute
  • to avoid duplication ,you typically use global styles

Learn more about all available CSS properties and values on the MDN CSS proprties reference
